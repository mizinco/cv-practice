# CIFAR-10 画像分類

PyTorchを使ったCNNによるCIFAR-10の画像分類実装です。

## 環境
- Python 3.11
- PyTorch 2.11.0
- torchvision

## モデル構造
- 畳み込み層3層（Conv2d）
- MaxPooling
- 全結合層2層

## 実験結果

| 条件 | 正解率 |
|------|--------|
| 2層・3epoch | 67.17% |
| 3層・3epoch | 68.50% |
| 3層・10epoch | 72.83% |

## わかったこと
畳み込み層を増やすよりも、epoch数を増やす方が正解率の向上に対する影響が大きかった。
